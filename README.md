# Boolean Retrieval Model - IR Assignment

This project implements a Boolean Retrieval Model using Python.

## How to Run

1. Extract the zip file
2. Make sure folder structure is same:
   - Trump Speechs folder
   - Stopword-List.txt
   - main.py

3. Run the program:
   python main.py

## Note
The program automatically reads files from the "Trump Speechs" folder.
